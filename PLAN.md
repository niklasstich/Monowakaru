I want to create a FF14 plugin that enables users to learn Japanese while playing the game.

One of the hardest challenges of learning Japanese is not being able to fully immerse in the language because reading it
requires extensive knowledge of Kanji. The more complex or adult the content becomes, the more this problem is exacerbated.

To make this easier, my ultimate goal is have a plugin where you can mouse-over any textbox in the game, press a hotkey
and the plugin will read the text from the textbox and offer dictionary entries for the exact text the user has hovered over.
The idea is heavily inspired by the browser addon Yomitan (previously Yomichan) and I want the general feel of the feature to be
similar to it in the finished product.

Just like in Yomitan, I want to be able to configure various different dictionaries which are shown in the tooltip window.
We should copy the dictionary format from Yomitan so we can use their dictionaries.
